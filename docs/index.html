Model Advisor

by @Jaace

Pick the right model before you burn a single token.

/plugin install model-advisor@Jaace-claude-model-advisor
user: Can you refactor the auth module to use JWT instead of sessions? ────────────────────────────────────────────────────────── Recommended: Sonnet / medium ── focused refactor, path is clear Run /model to switch before we start, or should I proceed?

How it works

  1. 1 Reads your first message
  2. 2 Assesses task complexity and the type of reasoning required
  3. 3 Recommends a specific model and effort level with a one-line rationale
  4. 4 Pauses — so you can switch models before any tokens are spent

Model guidance

Model Effort Best for
Haiku — Simple lookups, quick edits, boilerplate, single-fact questions
Sonnet
Coding, debugging, refactoring, explanations, any question with tradeoffs or nuanced judgment
Opus
Novel multi-system architecture, hard math, genuinely ambiguous open-ended design problems

Effort defaults to auto when not specified — Claude decides based on the task.

Mid-session reassessment

/model-check

Run this anytime during a session to reassess the current task and get an updated model and effort recommendation.

Install

Quick install
/plugin install model-advisor@Jaace-claude-model-advisor
Or add as marketplace source
/plugin marketplace add Jaace/claude-model-advisor
Update
/plugin marketplace update claude-model-advisor /plugin update model-advisor
